git简单教程

1.安装git：官网下载安装程序
2.在命令行输入：
$ git config –global user.name “Your Name”
$ git config –global user.email “email@example.com“
<设置自己在git上的name和email
3.创建库(repository):
命令行输入：
$ mkdir xxx（文件夹名字）
$ git init
<创建空目录，把这个目录变成git可以管理的仓库
4.把文件从工作区添加到暂存区
<工作区：电脑里看到的目录
<暂存区：stage或index
将文件放在仓库里后，命令行输入：
$ git add xxx.xx(文件名.后缀)
5.把暂存区的文件添加到版本库（master或main）
命令行输入：
$ git commit -m “xxx”(本次提交的说明)
<commit可以一次提交很多文件，可以多次add不同的文件
6.GitHub添加SSH KEY
7.连接GitHub上的远程仓库
在本地仓库下运行命令：
$ git remote add origin git@github.com:LeviDIAO/xxx(GitHub仓库名称).git
8.把本地库的所有内容推送到远程库上
$ git push -u origin master/main
<由于远程库是空的，第一次推送master/main分支时，加上了-u参数，Git不但会把本地的master分支内容推送的远程新的master分支，还会把本地的master分支和远程的master分支关联起来，在以后的推送或者拉取时就可以简化命令:
$ git push origin master
<把分支的最新修改推送到远程库上
